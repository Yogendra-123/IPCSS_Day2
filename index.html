1. What is box sizing?

In CSS, the CSS box-sizing property is used to specify how to calculate the total height and width of an element, It controls the size of an element with a specified height and width.

2. What is animation delay?

In CSS, the animation-delay CSS property specifies the amount of time to wait from applying the animation to an element before beginning to perform the animation.

3. Which property you will use to merge cells vertically in grid?

In CSS, for merge cells vertically we use grid-column.

4. Which property you will use to merge cells horizontally in grid?

In CSS, for merge cells vertically we use grid-row.

5. Explain CSS box model?

A box in CSS consists of a content area, which is where any text, images, or other HTML elements are displayed. This is optionally surrounded by padding, a border, and a margin, on one or more sides. The box model describes how these elements work together to create a box as displayed by CSS.

6. what is the difference between display none and visibility hidden?

In CSS, there is only one difference between display none and visibility hidden.
display:none removes the element as well as space in layout.
visibility:hidden hides the element, but it still takes up space in the layout.
